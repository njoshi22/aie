# RevOps Reconciliation Skills

## Available Actions

The RevMem service is authoritative for your role and allowed tools. The exact
tool set for a hosted run is generated by `GET /agents/{agent_id}/skill.md` and
the `allowed_tools` field returned by `GET /agents/{agent_id}` or `POST /agents`.

Common service tools include:

- **get_contract**: Fetch the signed contract / order-form pricing fields for a deal
- **get_crm_record**: Fetch the current CRM record for a deal
- **retrieve_context**: Query RevMem for past lessons and active policy rows
- **route_for_approval**: Route a material discrepancy for human approval
- **write_crm**: Request or apply a CRM correction through the service approval gate
- **get_approval_status**: Poll an approval request by `approval_request_id`
- **store_memory**: Persist a learned pattern or lesson to RevMem
